# ☀️ SunCart – Summer Essentials Store

SunCart is a modern summer-themed eCommerce web application where users can explore and purchase seasonal products such as sunglasses, summer outfits, skincare products, beach accessories, and more.

The application provides a smooth shopping experience with secure authentication, protected routes, responsive design, and profile management features.

**Live URL:** https://your-live-link.vercel.app

## 🎯 Project Purpose

The main purpose of this project is to create a modern and responsive summer eCommerce platform using Next.js and BetterAuth. Users can browse products, view detailed information, authenticate securely, and manage their profile.

---

## ✨ Key Features

* 🌅 Modern Summer Theme UI
* 🔐 Secure Authentication using BetterAuth
* 📧 Email & Password Login/Register
* 🔑 Google Social Login
* 🛡️ Protected Product Details Route
* 👤 User Profile Page
* ✏️ Update Profile Information
* 📦 Dynamic Product Details Page
* 📱 Fully Responsive Design
* 🎨 Beautiful Summer-inspired Interface
* 🎬 Lottie Animation Integration

## 📄 Pages & Routes

| Route             | Description                      |
| ----------------- | -------------------------------- |
| `/`               | Home Page                        |
| `/login`          | Login Page                       |
| `/register`       | Register Page                    |
| `/products/[id]`  | Product Details Page (Protected) |
| `/my-profile`     | User Profile Page                |
| `/update-profile` | Update User Information          |

## 🚀 Installation & Setup

### Clone Repository

```bash
git clone <repository-url>
```

### Move to Project Folder

```bash
cd suncart
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
